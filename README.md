## 사용 기술 
- `IDE` Sts
- `언어` Java 11
- `프레임워크 및 라이브러리`  Spring, BootStrap4, Maven
- `데이터베이스` Mysql 8.0
- `버전관리` Git
- `서버` NginX, Tomcat9.0
- `API` Kakao Map, Iamport
